About

Errantry is an open source adventure game development tool. It is internally developed at Meanfish Ltd to ease the development of educational point-and-click adventure games. The tool is an early alpha prototype that has been already been successfully used to create games such as Turvat for Aspa Foundation.

Current version is not recommended to be used by non-programmers or those unfamiliar with JavaScript. The prototype lacks validations which can potentially lead to obscure errors that can be very hard to fix. Additionally the modification of UI graphics and quest log requires the user to modify the source code by hand.

Installation

Installation for Windows

Errantry is easiest to install on Windows. Simply download the zip-package, extract everything and double-click the errantry.bat file from the root folder.

You do need to have Visual Studio 2012 redistributable installed for the accompanied php server to work. You can download it from here.

Installation for Mac OS X and Linux

Errantry requires a web server with PHP to work. Easiest way to get a server running without command line tools is to download a free Apache virtual server, such as XAMPP.
